# Tata Motors Credit Analysis

## Overview

This project presents a comprehensive credit analysis of Tata Motors using publicly available financial information. The analysis evaluates the company's credit profile through financial statement analysis, ratio analysis, cash flow assessment, and an indicative internal credit memorandum.

This project was developed to demonstrate practical credit analysis skills commonly used in corporate banking, credit risk, and commercial lending.
